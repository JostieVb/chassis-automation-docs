Front-end¶

Chassis Automation’s front-end was generated with Angular CLI version 1.7.0. Therefore, you need to install Node.js and Angular CLI if you don’t have it installed yet.

Build¶

Run ng build to build the project. The build artifacts will be stored in the dist/ directory. Use the -prod flag for a production build.
